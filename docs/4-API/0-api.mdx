## HTTP

```shell title="http-common.js"
import axios from "axios";

export default axios.create({
  baseURL: "http://localhost:8082/api",
  headers: {
    "Content-type": "application/json",
  },
});
```




#### /api/catcomp

- `GET` /api/catcomp/ - catcomp.findAll - Affiche toutes les catégories de compétences
- `GET` /api/catcomp/:id - catcomp.findOne - Affiche une catégorie de compétences par son ID

#### /api/classes

- `GET` /api/classes/ - classes.findAll - Affiche toutes les classes
- `GET` /api/classes/:id - classes.findOne - Affiche une classe

#### /api/elements

- `GET` /api/elements/ - elements.findAll - Affiche tous les éléments
- `GET` /api/elements/:id - elements.findOne - Affiche un élément

#### /api/eleves

- `GET` /api/eleves/ - eleves.findAll - Affiche tous les élèves
- `POST` /api/eleves/ - eleves.postOne - Ajoute un élève
- `GET` /api/eleves/:id - eleves.findOne - Affiche un élève
- `POST` /api/eleves/login - eleves.loginEleve - Connecte un élève
- `POST` /api/eleves/getUser - checkAuth - Vérifie l'autorisation d'un élève

#### /api/exercices

- `GET` /api/exercices/ - exercices.findAll - Affiche tous les exercices
- `GET` /api/exercices/:id - exercices.findOne - Affiche un exercice

#### /api/lessons

- `GET` /api/lessons/ - lessons.findAll - Affiche toutes les leçons
- `GET` /api/lessons/:id - lessons.findOne - Affiche une leçon

#### /api/matieres

- `GET` /api/matieres/ - matieres.findAll - Affiche toutes les matières
- `GET` /api/matieres/:id - matieres.findOne - Affiche une matière

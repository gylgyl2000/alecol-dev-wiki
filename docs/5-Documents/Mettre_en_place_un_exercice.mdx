# Mettre en place un exercice

## Choisir un type d'exercice

**Table :** type_exo

## Choisir une activité

**Table :** ac_clnt

Il est nécessaire que l'activité soit créée avec un `typrep` relatif au type d'exercice.

## Créer un exercice

**Table :** exercices

Utiliser le `pk_ac_id` de la table `ac_clnt` dans la colonne `fk_activite_id`.
